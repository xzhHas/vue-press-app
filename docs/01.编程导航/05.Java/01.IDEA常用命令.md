- 全文搜索：Ctrl+shift+A 或按两下 shift

- 复制整行：Ctrl+d

- 删除整行：Ctrl+y

- 向上方（下方）移动：alt+shift+上下键

- 收起（展开）一块代码：Ctrl+减号（等号）

- 展开全部代码：Ctrl+shift+等号

- 重命名字段：shift+F6

- 查看方法签名：Ctrl+p

- 快速查看文本光标处的文档：Ctrl+q

- 转到文件下一个高亮的错误：F2

- 展开警告说明：Ctrl+F1

- 在文件中查找并替换：Ctrl+shift+F

- 查看文件结构：Ctrl+F12

- 查看继承层次结构：Ctrl+alt+b 或 Ctrl+h 或 ctrl+shift+h
